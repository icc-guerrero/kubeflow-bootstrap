kubeflow-bootstrap

Kubeflow Bootstrap Helm Charts with FluxCD

A Helm chart for bootstrapping Kubeflow on Kubernetes using FluxCD GitOps principles. Deploy a complete ML platform with a single Helm install.

GitOps-Native FluxCD for continuous reconciliation
Modular Enable/disable individual components
Multi-Auth Dex, OAuth2 Proxy, OIDC support
Production Ready Proper dependency management

Usage

Helm must be installed to use the charts. Please refer to Helm's documentation to get started.

Once Helm is set up properly, add the repo as follows:

helm repo add kubeflow-bootstrap https://icc-guerrero.github.io/kubeflow-bootstrap

You can then run helm search repo kubeflow-bootstrap to see the charts.

Install

helm install kubeflow kubeflow-bootstrap/kubeflow-flux-bootstrap \
  --namespace kubeflow-system \
  --create-namespace \
  --values my-values.yaml

Available Charts

  • kubeflow-flux-bootstrap - Complete Kubeflow deployment using FluxCD
    Includes: Istio, cert-manager, Dex, Pipelines, Notebooks, KServe, and more

Documentation

Full documentation is available in the GitHub repository.

Prerequisites

  • Kubernetes cluster (v1.25+)
  • FluxCD installed on the cluster
  • Helm v3+
